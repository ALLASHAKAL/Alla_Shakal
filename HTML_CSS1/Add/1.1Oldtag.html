Тег <code> предназначен для отображения одной или нескольких строк текста, который представляет собой программный код.

example
X = X + 1

Контейнер <tt> отображает текст моноширинным текстом (эффект пишущей машинки или телетайпа). Этот тег относится к группе тегов физического форматирования.

Тег <kbd> обозначает пользовательский ввод: с клавиатуры, голосом, указателем или другим образом. например Ctrl+C;Ctrl+V; Delete

Тег <var> используется для выделения переменных компьютерных программ. Браузеры обычно помечают текст в контейнере курсивным начертанием.
